## 事前準備

  - [さくらのクラウドのコントロールパネル](https://secure.sakura.ad.jp/cloud/)からAPIキーを発行してください。

  - APIキーに対し「他サービスへのアクセス権」から `ウェブアクセラレータ`へのアクセス権を与えてください。

  - 作成したAPIキーの以下の値を控えておいてください。

    - アクセストークン
    - アクセスシークレット
